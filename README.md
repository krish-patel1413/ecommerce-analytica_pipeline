# E-Commerce Customer Analytics & Revenue Prediction Pipeline

##  Project Overview
This project simulates an enterprise-level e-commerce data pipeline. It extracts key performance indicators (KPIs) from a relational database and builds a machine learning regression model to predict customer lifetime value based on early transactional behavior.

##  Tech Stack & Skills Demonstrated
- **Languages:** Python, SQL
- **Data Infrastructure:** SQLite3 (Relational Schema Design & Transactions)
- **Data Manipulation:** Pandas (Data Aggregation & Joining)
- **Data Visualization:** Matplotlib (Object-Oriented Plotting)
- **Machine Learning:** Scikit-Learn (Linear Regression Modeling)

##  Analytical Insights & Core Implementation
1. **Database Simulation:** Programmatically generated mock customer records and localized sales transaction tables using Python constraints.
2. **RFM Analysis (SQL Queries):** Extracted the top 10 highest-spending customers and calculated total market revenues using aggregations (`SUM`, `COUNT`, `GROUP BY`) and relational `JOIN` functions.
3. **Market Visualization:** Engineered an optimized, scannable bar visualization tracking localized revenue performance across international markets (USA, Canada, India, etc.).
4. **Predictive Modeling:** Deployed a Linear Regression model to successfully forecast potential annual customer spending based on initial monthly purchase frequency metrics.
